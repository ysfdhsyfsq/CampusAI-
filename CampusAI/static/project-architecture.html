数据库设计

用户表 (user)
  • id: 用户ID (主键)
  • username: 用户名
  • password: 密码
  • role: 角色 (学生/商家/管理员)
  • phone: 手机号
  • email: 邮箱
  • create_time: 创建时间
商家表 (merchant)
  • id: 商家ID (主键)
  • name: 商家名称
  • description: 商家描述
  • logo: 商家Logo
  • status: 营业状态
  • user_id: 关联用户ID
  • create_time: 创建时间
菜品表 (dish)
  • id: 菜品ID (主键)
  • name: 菜品名称
  • price: 价格
  • image: 图片路径
  • description: 菜品描述
  • category_id: 分类ID
  • merchant_id: 商家ID
  • status: 上下架状态
自习室表 (study_room)
  • id: 自习室ID (主键)
  • name: 自习室名称
  • location: 位置
  • capacity: 容量
  • open_time: 开放时间
  • close_time: 关闭时间
  • status: 可用状态
座位表 (study_room_seat)
  • id: 座位ID (主键)
  • room_id: 自习室ID
  • seat_number: 座位号
  • status: 占用状态
  • user_id: 预约用户ID
  • reservation_time: 预约时间
订单表 (orders)
  • id: 订单ID (主键)
  • order_number: 订单编号
  • user_id: 用户ID
  • merchant_id: 商家ID
  • total_amount: 总金额
  • status: 订单状态
  • create_time: 下单时间
  • pay_time: 支付时间
活动表 (activity)
  • id: 活动ID (主键)
  • title: 活动标题
  • content: 活动内容
  • start_time: 开始时间
  • end_time: 结束时间
  • location: 活动地点
  • organizer: 组织者
  • status: 活动状态
